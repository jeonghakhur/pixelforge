## 2. Intentional Improvements (Positive Gaps)

| # | Plan | Implementation | Why Better |
|---|------|----------------|------------|
| 1 | diff2html code rendering | Direct highlight.js per-line `<table>` | Eliminated double line-spacing and `+` prefix artifacts |
| 2 | Reuse `Modal size="xl"` (white bg) | `createPortal` dark overlay | Matches `CodeViewModal` GitHub dark aesthetic — user-requested |
| 3 | `tokens` prop (current type only) | `getAllTokensAction()` for all types | Single unified CSS across color/typography/spacing/radius |
| 4 | Copy button only | Copy + Download (`.css` file) | Users can immediately save the file |
| 5 | `onCssPreview` prop callback from parent | Self-contained state in `TokenPageActions` | Better encapsulation, fewer props to thread |
