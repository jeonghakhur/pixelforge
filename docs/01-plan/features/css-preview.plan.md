# Plan: 토큰 페이지 CSS 미리보기 (diff2html + highlight.js)

## 1. 배경 및 문제

### 현재 구조의 한계

- 색상 토큰은 스워치 카드로만 표시 — CSS 변수 형식 코드 확인 불가
- "Export CSS" 버튼은 파일 다운로드 → 파일 열기 2단계 필요
- 특정 그룹(`brand/`, `neutral/`)만 선택해서 코드를 보고 싶어도 방법 없음
- 개발자가 토큰명 → CSS 변수명 매핑을 직관적으로 파악하기 어려움

### 원하는 상태

- 토큰 페이지에서 버튼 하나로 CSS 코드를 넓은 모달로 확인
- GitHub / diff2html UI처럼 — 라인 번호, 파일명 헤더, 구문 강조
- highlight.js CSS 언어팩으로 정확한 CSS 구문 강조
- 전체 복사 버튼 1회 클릭으로 완료
- color 페이지 우선 구현, 동일 패턴으로 typography / spacing / radius 확장 가능 설계

## 2. 라이브러리 선택

### 2-1. diff2html `v3.4.56`

**역할**: 코드 뷰어 UI (라인 번호, 파일 헤더, GitHub 스타일 레이아웃)

**사용 방식**: diff2html은 unified diff 포맷을 입력으로 받음.
CSS 코드를 "새로 추가된 파일"처럼 가상 diff로 래핑하여 렌더링:
```
--- /dev/null
+++ b/tokens.css
@@ -0,0 +1,N @@
+/* Generated by PixelForge */
+
+:root {
+  --color-brand-primary: #007AFF;
...
```
→ diff2html이 이를 파일 추가 뷰로 렌더링 (모든 라인이 초록 `+` 없이, `side-by-side` 대신 `line-by-line` 사용)

**실제로 사용할 옵션**:
```typescript
import { Diff2Html } from 'diff2html';

const html = Diff2Html.getPrettyHtml(diffText, {
  inputFormat: 'diff',
  showFiles: true,          // 파일명 헤더 표시
  matching: 'lines',
  outputFormat: 'line-by-line',  // 단일 컬럼 (사이드바이사이드 X)
  highlight: true,          // highlight.js 연동
});
```

### 2-2. highlight.js `v11.11.1`

**역할**: CSS 구문 강조

**사용 방식**: diff2html의 `highlight: true` 옵션과 자동 연동.
CSS 언어팩만 개별 import (번들 최소화):
```typescript
import hljs from 'highlight.js/lib/core';
import css from 'highlight.js/lib/languages/css';
hljs.registerLanguage('css', css);
```

**테마**: `highlight.js/styles/github-dark.css` (GitHub 다크 코드 블록 스타일)

### 2-3. 패키지 추가

```bash
npm install diff2html highlight.js
```

## 3. 기능 명세

### 3-1. CSS 미리보기 모달 (`CssPreviewModal`)

**트리거**: 페이지 헤더 액션 영역의 "CSS 보기" 버튼 (`TokenPageActions.tsx`)

**모달 구조** (`size="xl"`, 기존 `Modal` 컴포넌트 재사용):
```
┌──────────────────────────────────────────────────────────────────┐
│  CSS Variables                          [전체 복사] [✕]          │  ← Modal 헤더
├──────────────────────────────────────────────────────────────────┤
│ ┌────────────────────────────────────────────────────────────┐   │
│ │ 📄 tokens.css                                              │   │  ← diff2html 파일 헤더
│ ├──────────────────────────────────────────────────────────  │   │
│ │  1 │ /* Generated by PixelForge · 2025-03-24 */           │   │
│ │  2 │                                                       │   │  ← 라인 번호
│ │  3 │ :root {                                               │   │
│ │  4 │   /* brand */                                        │   │
│ │  5 │   --color-brand-primary:         #007AFF;            │   │  ← highlight.js 강조
│ │  6 │   --color-brand-primary-hover:   #0066D6;            │   │
│ │    │   ...                                                 │   │
│ │ N  │ }                                                     │   │
│ └────────────────────────────────────────────────────────────┘   │
└──────────────────────────────────────────────────────────────────┘
```

**코드 영역**: `overflow-y: auto`, 최대 높이 `65vh`

### 3-2. CSS 변수명 생성 규칙

- 토큰명 `brand/primary` → `--color-brand-primary`
- 토큰명 `neutral/100` → `--color-neutral-100`
- 슬래시(`/`) → 하이픈(`-`), 공백 → 하이픈, 소문자 정규화
- 타입 prefix: color → `--color-`, typography → `--font-`, spacing → `--spacing-`, radius → `--radius-`
- 그룹별 `/* group-name */` 구분 주석 자동 삽입

### 3-3. "CSS 보기" 버튼 (`TokenPageActions` 확장)

- 기존 버튼 목록에 "CSS 보기" 버튼 추가
- 아이콘: `solar:code-linear`
- 클릭 시 `onCssPreview()` prop 콜백 호출

## 5. CSS 코드 출력 포맷

```css
/* Generated by PixelForge · 2025-03-24 */

:root {
  /* brand */
  --color-brand-primary:         #007AFF;
  --color-brand-primary-hover:   #0066D6;
  --color-brand-secondary:       #5856D6;

  /* neutral */
  --color-neutral-50:            #FAFAFA;
  --color-neutral-100:           #F5F5F5;
  --color-neutral-900:           #171717;

  /* semantic */
  --color-success:               #34C759;
  --color-warning:               #FF9500;
  --color-danger:                #FF3B30;
}
```

**정렬 규칙**: 값 부분을 컬럼 정렬 (가장 긴 속성명 기준 공백 패딩)

## 6. diff2html 스타일 오버라이드

diff2html 기본 테마를 PixelForge Vantablack Luxe에 맞게 오버라이드:

```scss
// css-preview.module.scss
// diff2html의 "추가" 라인 초록색을 중립색으로 덮어씀 (단순 코드 뷰)
.codeViewer {
  :global(.d2h-file-wrapper) { border: 1px solid var(--glass-border); border-radius: $border-radius-lg; }
  :global(.d2h-file-header)  { background: var(--bg-surface); color: var(--text-secondary); }
  :global(.d2h-code-line)    { background: var(--bg-elevated) !important; }  // +라인 초록 제거
  :global(.d2h-ins)          { background: var(--bg-elevated) !important; color: var(--text-primary) !important; }
  :global(.d2h-ins .d2h-code-line-ctn) { background: none !important; }
  :global(.line-num1)        { color: var(--text-muted); font-size: $font-size-sm; }
}
```
